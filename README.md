 Fanta Concept Landing Page

Este projeto é uma Landing Page interativa e animada, desenvolvida como objeto de estudo para aprofundar conhecimentos em **JavaScript** e na implementação da biblioteca de animação **GSAP (GreenSock Animation Platform)**.

Objetivo do Projeto

O foco principal deste projeto foi sair do básico do CSS e criar uma experiência imersiva controlada pelo scroll do usuário (ScrollTrigger), manipulando elementos do DOM em tempo real.

ecnologias Utilizadas

- **HTML5** (Estrutura semântica)
- **CSS3** (Estilização, Flexbox e Posicionamento absoluto)
- **JavaScript** (Lógica de manipulação do DOM)
- **GSAP** (Biblioteca de animação)
  - `gsap.timeline()`: Para sequenciar as animações.
  - `ScrollTrigger`: Para disparar animações baseadas na posição da barra de rolagem.

Funcionalidades e Aprendizados

- **Animação Sincronizada com Scroll:** A lata de Fanta e as laranjas se movem e giram conforme o usuário desce a página, transitando entre a seção "Hero" e a seção de informações.
- **Timelines do GSAP:** Criação de linhas do tempo complexas para garantir que múltiplos elementos (folhas, frutas, latas) se movam em harmonia.
- **Transição de Seções:** A lata principal "viaja" da primeira seção até se encaixar perfeitamente no card de produtos da terceira seção.
- **Design Responsivo:** Ajustes de layout utilizando unidades relativas (`vh`, `vw`, `%`).

Como visualizar

1. Clone o repositório.
2. Abra o arquivo `index.html` no seu navegador.
3. Role a página para ver as animações em ação!
